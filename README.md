## Project setup

```bash
$ npm install
```

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

Node: v22.14.0

Setup:

1. clone the repo.
2. run npm i with using node 22
3. Create a .env file using the example env with correct db creds
4. start the app using run command
